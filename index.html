SimpleDB Documentation

Introduction

SimpleDB is a lightweight, file-based NoSQL database designed for simplicity and ease of use. It supports basic operations such as creating tables, inserting, updating, and deleting records. This document provides a comprehensive guide on how to use SimpleDB effectively.

Getting Started

To get started with SimpleDB, you first need to install the package and set up your environment. Below is a basic example:

const { SimpleDB, Schema, Model } = require('simple.dbms');
const path = require('path');

// Initialize SimpleDB
const db = new SimpleDB(path.resolve(__dirname, 'database.spdb'));

// Define a schema
const userSchema = new Schema({
    userId: { type: 'string', unique: true },
    balance: { type: 'number', default: 0 }
});

// Create a model
const UserModel = new Model('users', userSchema, db);

Schema Definition

A schema defines the structure of the data you store in your database. Each schema consists of fields and their respective types. Here's how to define a schema:

const userSchema = new Schema({
    userId: { type: 'string', unique: true },
    balance: { type: 'number', default: 0 }
});

Basic Operations

Creating Records

To create a new record, use the create method:

const newUser = { userId: '12345', balance: 100 };
await UserModel.create(newUser);

Finding Records

To find records, use the find or findOne methods:

const user = await UserModel.findOne({ userId: '12345' });
console.log(user); // Output: { userId: '12345', balance: 100 }

Updating Records

To update a record, use the update method:

await UserModel.update({ userId: '12345' }, { balance: 200 });

Deleting Records

To delete a record, use the delete method:

await UserModel.delete({ userId: '12345' });

Advanced Features

Auto-Increment Fields

You can define auto-incrementing fields in your schema:

const autoIncrementSchema = new Schema({
    id: { type: 'number', autoIncrement: true },
    name: { type: 'string' }
});

Unique Fields

To ensure that a field value is unique, set the unique property:

const userSchema = new Schema({
    email: { type: 'string', unique: true }
});
